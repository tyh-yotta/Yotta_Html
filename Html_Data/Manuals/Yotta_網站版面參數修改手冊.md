# Yotta 網站版面參數修改手冊

本手冊整理了您最常需要修改的幾個**網站全域版面參數**，例如：最上方的跑馬燈公告、上方主選單（導覽列），以及側邊欄的產品類別與子項目。

這些設定皆採用「資料驅動」的方式管理，您只要修改指定的 JavaScript 檔案中的文字，全網站就會自動同步更新，不需要一頁一頁去改 HTML！

---

## 一、如何修改「最上方跑馬燈公告文字」？

**檔案位置：** `message.js` (位於專案根目錄)

這是專門用來管理全站頂部公告欄（跑馬燈）的檔案。

### 修改方法：
打開 `message.js`，您會看到以下內容：
```javascript
const siteMessage = "★ 2026 最新消息：全新 A-1080 模組即將上市，歡迎來信預約測試！ ★";
```

- **更改文字**：直接修改雙引號 `""` 裡面的文字即可。
- **關閉跑馬燈**：如果您暫時沒有要公告的消息，請將引號內的文字清空（保留雙引號），跑馬燈就會自動消失並隱藏起來。
  ```javascript
  const siteMessage = "";
  ```

## 二、如何修改「上方主選單 (導覽列)」？

**包含項目：** 首頁、關於我們、最新消息、技術支援、產品介紹、交通位置、聯絡我們
**檔案位置：** `js/data.js` (請往下捲動尋找 `mainMenuData`)

主選單的結構儲存在 `mainMenuData` 陣列中，每一個大括號 `{}` 就是一顆按鈕。

### 修改方法：
```javascript
const mainMenuData = [
    { id: "home", title_zh: "首頁", title_en: "Home", link: "index.html" },
    { id: "about", title_zh: "關於我們", title_en: "About Us", link: "about/about.html" },
    // ... 可以依照需求新增或刪除
];
```

- **修改名稱**：更改 `title_zh` 的值可修改中文名稱，更改 `title_en` 可修改切換成英文版時的名稱。
- **修改連結網址**：更改 `link` 的值，決定使用者點擊後會連到哪個網頁。
- **新增選單**：直接複製其中一行大括號 `{...}`，貼在後面，並更改裡面的參數即可（別忘了上一行的結尾要加逗號 `,`）。
- **調整順序**：直接把整行 `{...}` 剪下貼上到您想要的位置，就可以改變選單在畫面上由左至右的排序！

## 三、如何修改「左側產品類別與子項目」？

**包含項目：** SCADA-IPC、工業電腦、I/O 模組、控制器 等大分類與其底下的型號。
**檔案位置：** `js/data.js` (請尋找 `categoryData`)

側邊選單分為「主分類 (Group)」與「子項目 (Items)」兩層結構。

### 修改方法：
```javascript
const categoryData = [
    {
        groupId: "io-modules",
        title_zh: "I/O 模組",
        title_en: "I/O Modules",
        icon: "expand_less",
        isOpen: true,  // 預設是否展開：true 為展開，false 為折疊
        items: [
            { title_zh: "A-10x系列(RS485)", title_en: "A-10x Series (RS485)", link: "products/products_a10x.html" },
            { title_zh: "A-12x 系列(WIFI)", title_en: "A-12x Series (WIFI)", link: "#" }
            // ... 子項目列表
        ]
    }
];
```

#### 📌 主分類修改：
- `title_zh` / `title_en`：修改大分類的中英文名稱（例如：I/O 模組）。
- `isOpen`：若設為 `true`，使用者進到網頁時這個分類就會預設為「展開」狀態；若為 `false` 則預設為折疊。

#### 📌 子項目修改 (`items` 陣列內部)：
- 在 `items: [...]` 的中括號裡，每一行 `{}` 都是一個次要選項。
- 您可以自由修改 `title_zh` 來改變名稱，並修改 `link` 來決定點擊後要前往哪個產品頁面。
- **若某個子系列尚未建置網頁**：請將 `link` 設定為 `"#"`，這樣使用者點擊時就不會亂跳轉。

---

## ⚠️ 撰寫與修改時的注意事項（重要防錯）

1. **標點符號**：請務必確保所有的引號 `""`、逗號 `,`、大括號 `{}` 都是**半形英文符號**。如果打成全形中文字元，網頁會壞掉無法顯示！
2. **逗號規則**：
   - 陣列（`[` 與 `]`）或物件（`{` 與 `}`）裡面的**每一項之間，必須要有逗號分隔**。
   - 但是**最後一項的結尾，建議不要加上逗號**。
3. **測試與驗證**：修改完畢存檔後，只需重新整理瀏覽器網頁（F5），修改的內容就會立即生效，且全站所有分頁都會自動同步更新。
